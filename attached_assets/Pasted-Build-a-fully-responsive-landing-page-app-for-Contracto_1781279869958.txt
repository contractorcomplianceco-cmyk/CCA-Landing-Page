Build a fully responsive landing page app for Contractor Compliance Authority (CCA) in Node.js + Express. Use the following specifications:

1. Serve the landing page at "/" using the HTML, CSS, and JS below.
2. Keep the exact CCA branding from https://www.contractor-compliance-authority.com/:
   - Colors: dark/navy background #0A1F2F, teal accent #00B5AD, white text
   - Fonts: clean sans-serif (Arial or system font)
   - Header and footer styling consistent with the website
   - Logo included in header and footer
3. Page sections:
   - Hero with headline: "Contractor Licensing & Compliance Support You Can Trust"
   - Subheadline: "Get clear next steps, organized compliance, and risk‑managed licensing — without the guesswork."
   - CTA button: "Schedule My Compliance Review", scrolls to form
   - Services/Why Choose CCA section
   - How It Works / Step-by-Step section
   - Core Benefits
   - Who We Serve
   - Credibility / Proof points
   - Footer with contact, privacy, terms
4. Form Section:
   - Fields: Full Name, Email, Phone, Company, Service Needed (dropdown), State (dropdown)
   - On submit, log the data to console and display a thank-you message
   - Smooth scroll to form on CTA click
5. Responsive layout for mobile, tablet, desktop
6. Minimal button hover animations (scale and color transition)
7. SEO-friendly title and meta description
8. No external frameworks required besides Express and body-parser

**Deliverables / Folder Structure:**
- server.js: Express server serving the landing page and handling form submission
- public/cca_landing.html: full HTML page
- public/css/styles.css: CSS styling
- public/js/main.js: JS for form submission and animations
- package.json: with Express and body-parser dependencies
